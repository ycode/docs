# Ycode Cloud

Ycode Cloud is the managed hosting platform for Ycode. Sign up, create projects, and start building — no server setup required. Ycode Cloud handles hosting, deployments, SSL, and infrastructure.

## Dashboard

The dashboard is your home in Ycode Cloud. It shows all your projects as cards with thumbnails, names, and URLs. Click a project to open it in the editor.

From the dashboard you can:

- **Create new projects** — Start from scratch, choose a template, or import from Ycode Legacy
- **Open projects** — Click a card to launch the editor
- **Manage billing** — Access the Stripe customer portal from the project menu
- **Delete projects** — Remove projects you no longer need
