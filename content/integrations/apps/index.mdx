# Apps

Connect Ycode with third-party applications. App integrations let you send form data, sync content, and automate workflows with external services.

## Available Apps

### Airtable

One-way sync from Airtable tables to your CMS collections. Map fields between Airtable and your CMS, run manual syncs, and enable auto-sync via webhooks for real-time updates.

[Airtable integration](/docs/integrations/apps/airtable)

### MailerLite

Connect forms to MailerLite subscriber groups. When a form is submitted, new subscribers are added automatically with mapped field data.

[MailerLite integration](/docs/integrations/apps/mailer-lite)

### Google Maps

Provide a Google Maps API key for Map elements using the Google Map provider. Requires the Maps Embed API and Places API enabled on your Google Cloud project.

[Google Maps integration](/docs/integrations/apps/google-maps)

### Mapbox

Provide a Mapbox access token for Map elements using the Mapbox provider. Supports five map styles, custom marker colors, and navigation controls.

[Mapbox integration](/docs/integrations/apps/mapbox)

## Coming Soon

Additional app integrations are planned for future releases, including Mailchimp, Zapier, and Make.
